READ ME FIRST

This is simple Legend of Zelda inspired game coded in Python by me with the assistance of AI.
The game utilizes dictionaries, classes, functions, tuples, and various other concepts I learned in CS 223 Programming for IT. 

The game can be run by cloning or downloading the zip file and running "main.py". As long as pygame has been installed on the system everything should work as expected.

Enjoy the game!
